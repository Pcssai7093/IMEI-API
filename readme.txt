commands and info

RUN the server:
run the index.js file

To validate IMEI:
go to localhost:3000/VerifyIMEI/Your_IMEI

note:
1)To make testing easy IMEI is passed in URL for validation.
2)Give input IMEI as a parameter in place of "Your_IMEI".
3)On sending a request validation response will be displayed on the browser.

example request after running the server
http://localhost:3000/VerifyIMEI/866407051749463
